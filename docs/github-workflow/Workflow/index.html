Default values and combinators

The following functions give default values, in essense this means lists are set to empty lists and optional values are set to None. Where this isn't possible the function will have arguments to provide the details to construct the values.

val simple_event : string list -> Types.on

Construct simple triggers for the workflow as a list such as [push, pull_request]

val complex_event : Events.t -> Types.on

Construct a complex trigger for workflows using events
